# Object-oriented

将数据块及相关行为封装为由Class生成的Object

多个类组织形成Class hierarchies

# OOP基础

  • Abstraction：反应特定内容详细信息且忽略其他的模型
  • Encapsulation：Object对其他Object隐藏部分状态和行为且暴露有限接口的能力
  • Inheritance：根据已有Class创建新Class的能力
  • Polymorphism：在当前Context不知道Object真实类型时检测其所属真实Class并调用其实现的能力

# Object之间的关系

  • Dependency：修改B影响A

    局部变量、形参

    • Association：C知道D

      • Aggregation：E知道F且由F构成

        has-a，语义加强

          is-a，语义最强

    • Implementation：I定义的方法由J声明
      • Inheritance：K继承L的和实现且可以扩展
